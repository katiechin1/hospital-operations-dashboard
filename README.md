# Hospital Patient Flow & Efficiency Dashboard
 
### Overview

Analyze hospital patient flow data to uncover scheduling and process inefficiencies.
The goal: reduce wait times, improve on-time arrivals, and optimize resource allocation without increasing staff headcount.

### Data Set
- Patient-level hospital wait time and arrival data
- Department staffing and occupancy records
- Source: [Hospital Wait Time Data – Kaggle](https://www.kaggle.com/datasets/bharathreddybollu/hospital-wait-time-data/data)

**Key Documents**
- [Data Audit](./data_audit.md) — data quality checks, cleaning steps, and outlier handling  
- [KPI Definitions](./kpis.md) — metrics, formulas, and realistic performance targets  
- [Insights & Recommendations](./insights.md) — stakeholder-oriented findings and next steps
- [KPI Documentation](./kpis.md) - KPI definitions and calculation logic

### Tools 
- **Microsoft Excel** (PivotTables, Slicers, Conditional Formatting)  
- **Power Query** for data cleaning and transformation  
- **Data Analysis** (KPI definition, benchmarking, process metrics)  
- **Dashboard Design** (KPI cards, filters, and performance visuals)
