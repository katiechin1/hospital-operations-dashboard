# Hospital Patient Flow & Efficieny Dashboard

### Business Task

Analyze hospital patient flow data to evaluate operational efficiency across departments, focusing on wait times, arrival punctuality, and visit duration.
The goal is to identify process bottlenecks and recommend actionable workflow improvements to enhance patient experience.

### Data Set

- Patient visit data containing arrival times, triage completion, provider start times, and department assignment

- Department-level staffing and occupancy metrics

- Source: [Hospital Wait Time Data - Kaggle](https://www.kaggle.com/datasets/bharathreddybollu/hospital-wait-time-data/data)

### Tools

- Microsoft Excel (Power Query) – for data cleaning, transformation, and model building

- Excel Dashboards – for visualization, KPI tracking, and analysis presentation
